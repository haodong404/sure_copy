Rust code in `sure_copy_core` favors small modules, explicit domain structs/enums, and builder-style `with_*` methods. Errors are represented by a structured `CopyError` with category/code/message rather than `anyhow`. Async traits are implemented with `async-trait`; Tokio is the async runtime; SQL persistence uses `sqlx` with SQLite. Frontend formatting is Biome with 2-space indentation. Naming is descriptive and API-facing docs are written as Rust doc comments.

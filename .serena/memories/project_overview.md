SureCopy is a Rust + Tauri + SolidJS workspace. The root Cargo workspace contains `src-tauri` and `sure_copy_core`; the frontend uses Vite/SolidJS. `sure_copy_core` is the Rust core module for copy-task orchestration, copy pipelines, durable task state, and filesystem/checksum abstractions. The codebase is layered into `domain`, `orchestrator`, `pipeline`, `infrastructure`, and `interface`. README emphasizes correctness, recoverability, extensibility, observability, and controllable performance.
